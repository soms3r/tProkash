# Bangladesh Publishing Handbook & Open Publisher Directory

> The most comprehensive open knowledge base for the Bangladeshi publishing ecosystem.

---

## Vision

To build an open, accurate, and community-maintained knowledge base of the Bangladeshi publishing industry.

This project aims to help:

- Authors
- Publishers
- Editors
- Designers
- Translators
- Booksellers
- Researchers
- Self-publishing startups

---

## Goals

- Create a verified directory of Bangladeshi publishers.
- Document the entire publishing workflow.
- Provide practical resources for first-time authors.
- Publish structured open datasets.
- Build an API-ready database.

---

## What's Included

- Publisher Directory
- Printing Directory
- Editor Directory
- Translator Directory
- Bookstore Directory
- Publishing Handbook
- Legal Resources
- Contracts
- Templates
- Marketing Guides
- Open Data
