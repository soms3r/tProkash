# Scripts

## What belongs here

Automation scripts, utility programs, and workflow documentation for data import, export, validation, and maintenance tasks. Workflow documentation files (*.md) describe procedures; actual script files (*.py, *.sh, *.ps1) implement them.

## What does NOT belong here

- Application source code (see `backend/`)
- Database schemas (see `database/`)
- Data files (see `data/`)

## Dependencies

- Import workflow: `scripts/import-workflow.md`
- Import pipeline: `docs/IMPORT_PIPELINE.md`
- Export pipeline: `docs/EXPORT_PIPELINE.md`
