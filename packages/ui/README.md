# UI

Shared UI component library for tProkash applications.

**Stack:** TypeScript, React, Tailwind CSS

**What it contains:**
- Design system primitives (Button, Input, Card, Badge, Tag, Modal)
- Layout components (Header, Footer, Sidebar, Container)
- Domain-specific components (PublisherCard, BookCard, AuthorCard, SearchBar, FacetedFilters)
- Verification badge component
- Bilingual display components

**Depends on:** `@tprokash/types`, `@tprokash/utils`

**Used by:** `apps/web`, `apps/admin`
