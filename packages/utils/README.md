# Utils

Shared utility functions used across the tProkash monorepo.

**Stack:** TypeScript

**What it contains:**
- ID generation and parsing (prefixed IDs: PUB-, BOK-, PER-)
- Date/time formatting and localization
- String utilities (slugify, truncate, bilingual helpers)
- Number formatting
- Validation helpers (email, URL, phone, ISBN)
- Array and object utilities

**Depends on:** None

**Used by:** All packages, apps, and services
