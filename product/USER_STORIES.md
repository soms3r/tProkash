# User Stories — v0.2

## Author

1. As an author, I want to search publishers by genre so that I can find publishers relevant to my manuscript.
2. As an author, I want to view a publisher's full profile including submission guidelines so that I know how to submit my work.
3. As an author, I want to see which publishers have published books in my language so that I can target my submissions.
4. As an author, I want to view my author profile with my published books listed so that readers can discover my work.
5. As an author, I want to search for books in my genre so that I can understand the competitive landscape.
6. As an author, I want to filter publishers by their verification status so that I know the data is trustworthy.

## Publisher

7. As a publisher, I want to view my publisher profile exactly as the public sees it so that I can verify the information is accurate.
8. As a publisher, I want to search for other publishers by service type so that I can find potential partners.
9. As a publisher, I want to browse books in my category so that I can track market trends.
10. As a publisher, I want to download the publisher directory as a CSV so that I can analyze the data in spreadsheets.
11. As a publisher, I want to see which imprints are listed under my publishing house so that our brand structure is accurately represented.
12. As a publisher, I want to access the API documentation so that I can integrate tProkash data into my own systems.

## Reader

13. As a reader, I want to search books by title, author, or category so that I can find books I am interested in.
14. As a reader, I want to view a book's detail page including description and ISBN so that I can find and purchase the book.
15. As a reader, I want to see an author's complete bibliography so that I can discover more books by authors I enjoy.
16. As a reader, I want to toggle between Bengali and English so that I can read content in my preferred language.
17. As a reader, I want to see verification badges on profiles so that I know the information has been verified.
18. As a reader, I want autocomplete suggestions when typing in the search bar so that I can find results faster.
19. As a reader, I want to browse books by category so that I can discover new books in genres I enjoy.
20. As a reader, I want to see related books on a publisher's page so that I can discover more from the same publisher.

## Researcher

21. As a researcher, I want to download the full publisher dataset in JSON format so that I can perform statistical analysis.
22. As a researcher, I want to filter search results by verification level so that I can use only verified data in my research.
23. As a researcher, I want to access the API so that I can query the data programmatically.
24. As a researcher, I want to see source attribution on data records so that I can verify the provenance of the information.
25. As a researcher, I want to view the total number of publishers, books, and authors in the directory so that I can cite aggregate statistics.
26. As a researcher, I want to search across all entity types (publishers, books, authors) so that I can find any information from a single search.

## Bookseller

27. As a bookseller, I want to search for active publishers in my region so that I can source books.
28. As a bookseller, I want to view publisher contact information and address so that I can place orders.
29. As a bookseller, I want to see which publishers are no longer active so that I do not waste time contacting defunct businesses.
30. As a bookseller, I want to browse publishers by the services they offer so that I can find distributors and wholesalers.

## Administrator

31. As an administrator, I want to see verification status badges on all profiles so that I can quickly assess data quality.
32. As an administrator, I want to search for entities with low verification levels so that I can prioritize data improvement efforts.
33. As an administrator, I want to access the API documentation to verify endpoint behavior matches the specification.
34. As an administrator, I want to view the dataset download page so that I can confirm dataset packages are correctly published.
35. As an administrator, I want to view all entity types (publishers, books, authors) in a consistent format so that I can verify uniformity.
