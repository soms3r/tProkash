# Minimum Viable Product — v0.2

## MVP Scope

The v0.2 MVP is a **read-only public directory** of Bangladeshi publishers, books, and authors with full-text search and downloadable datasets.

## In scope

### Data
- 50+ verified publisher profiles with contact info, services, imprints, and status
- 200+ book records with titles (bn/en), authors, publishers, categories, ISBNs
- 100+ author profiles with biography and bibliography
- All data bilingual (Bengali + English)
- Every record carries verification status and source attribution

### Pages
- Home page: featured content, stats, search bar, quick links
- Search page: full-text search across publishers, books, authors with filters
- Publisher profile: full detail page
- Book profile: full detail page with edition info
- Author profile: full detail page with bibliography
- About page: project background, license, team
- Datasets page: downloadable open data packages
- API page: documentation and usage guide
- 404 page

### Features
- Full-text search with autocomplete
- Faceted filtering by category, language, status, verification level
- Publisher-to-book and book-to-author navigation
- Bilingual display (toggle bn/en)
- Dataset downloads (JSON, CSV)
- Public read-only REST API (versioned)
- Responsive design (mobile + desktop)
- Basic analytics (page views, search queries)

## Out of scope (v0.2)

- User authentication and accounts
- Data submission or editing
- Reviews, ratings, comments
- Publishing workflow (submissions, contracts)
- Printing, distribution, bookstore directories
- Admin panel or dashboards
- AI-powered features
- Marketplace or e-commerce
