# Product

This directory contains the implementation specification for tProkash **Version 0.2 (Milestone 2, Sprint 1)** .

## What is here

| File | Purpose |
|------|---------|
| `VISION.md` | Product vision for v0.2 |
| `MVP.md` | Minimum Viable Product scope |
| `FEATURE_MATRIX.md` | Feature classification (Now/Next/Later) |
| `USER_PERSONAS.md` | Six user personas |
| `USER_STORIES.md` | Practical user stories by persona |
| `NAVIGATION.md` | Public page specifications |
| `SEARCH_SPECIFICATION.md` | Search behavior and field definitions |
| `PUBLIC_PROFILE_SPEC.md` | Publisher, Book, Author profile field specs |
| `ROADMAP_V0.2.md` | Release roadmap for v0.2 |

## How to use

These documents drive implementation during Sprint 1. Every feature, page, and data point described here must be built before the v0.2 release. Architects use these specs to create ADRs; engineers use them to write code; testers use them to verify completeness.
