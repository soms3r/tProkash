# Web

Public-facing web application for tProkash — the main user-facing product.

**Stack:** TypeScript, React, Next.js

**Purpose:** Browse and search publishers, books, authors; download datasets; view API docs.

**Depends on:** `@tprokash/ui`, `@tprokash/types`, `@tprokash/utils`, `@tprokash/validation`

**Consumes:** `apps/api` (via HTTP), `services/search`
