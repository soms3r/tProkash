# Admin

Internal administration dashboard for tProkash editors and moderators.

**Stack:** TypeScript, React, Next.js

**Purpose:** Data management, verification workflows, user management, system configuration.

**Depends on:** `@tprokash/ui`, `@tprokash/types`, `@tprokash/utils`, `@tprokash/validation`

**Consumes:** `apps/api` (via HTTP), `services/search`, `services/notifications`
